De iconen kunnen direct in de HTML gebruik worden of via CSS. Voor gebruik in de HTML moet de karaktercode vooraf gegaan worden door &#x en worden afgesloten met een ;

Wanneer een icoon middels CSS wordt gebruikt dan dient de karaktercode vooraf gegaan te worden met een \
