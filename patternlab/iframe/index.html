                        <iframe data-module="ui/iframe-medicinfo" src="http://medicinfo.cz.nl/medischeencyclopedie" name="dyniframe_{bf0bb88a-3e66-4c42-8b60-8371bb6cd7a4}_@site.iframeBase"
                              id="dyniframe_{bf0bb88a-3e66-4c42-8b60-8371bb6cd7a4}_@site.iframeBase"
                              frameborder="0" height="0"></iframe>

src

De src attribuut is de pagina die getoond wordt.

name & id

De name en id moeten gelijk zijn aan elkaar. De name wordt gebruikt in het script om de hoogte van de pagina op te vragen, maar de id is belangrijk als er meerdere iframes op de CZ pagina staan om de hoogte aan te passen. Het zal bijna niet voorkomen dat er meer dan een iframe op de site zal staan.

De opbouw van de name en id waarde is ook specifiek. Zo begint het met dyniframe_{ gevolgd door de id van de pagina bij Medicinfo. Deze id is optioneel. Het eindigt met een }_http://www.cz.nl. De URL wordt gebruikt om te controleren dat de communicatie tussen de CZ pagina en de Medicinfo pagina is, anders zal er geen informatie worden uitgewisseld.

Het id is van belang als er meerdere iframes op een pagina staan. Dan gebruikt het script de id van het frame om te weten welk frame in hoogte aangepast moet worden.
