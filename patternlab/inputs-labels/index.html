Wrap inputs

De formulierstyling zegt dat inputs onder het bijbehorende label dienen te staan. Hierdoor is het mogelijk om de input altijd te wrappen in het label. Dit heeft nog een aantal andere voordelen die verderop besproken zullen worden.


    <form action="">
        <label for="test1">
            Dit is het label van onderstaande input
            <input type="text" id="test1">
        </label>
        <label for="test2">
            Dit is een label met een medium size input
            <input type="text" id="test2" class="medium-size">
        </label>
        <label for="test3">
            Dit is een label met een small size input
            <input type="text" id="test3" class="small-size">
        </label>
    </form>

Zorg er voor dat een label altijd een geldige waarde in het for-attribuut heeft staan. Dit dient altijd te refereren aan het input element dat erbij hoort. Screenreaders hebben deze informatie nodig om te weten welk label er voorgelezen moet worden.

Er zijn drie maten beschikbaar voor de inputs:

  • Full size, hier is geen extra class voor nodig. Standaard neemt een input de volledige breedte in.
  • Medium size, hier is de class medium-size voor nodig. Het input veld zal nu 60% van de beschikbare breedte innemen. Wanneer de breedte van de viewport in de breakpoints XS of S valt zal de input 100% breed zijn.
  • Small size, hier is de class small-size voor nodig. Het input veld zal nu 30% van de beschikbare breedte innemen. Wanneer de breedte van de viewport in de breakpoints XS of S valt zal de input 100% breed zijn.

Verplichte velden

Verplichte velden worden niet extra gemarkeerd met tekst daar we uitleggen dat standaard alle invulvelden verplicht zijn. Wel dienen deze velden een extra aria attribuut te krijgen, te weten aria-required="true".


    <form action="">
        <label for="test4">
            E-mailadres
            <input type="email" id="test4" class="medium-size" aria-required="true">
            <div class="sub-label">We gebruiken uw e-mailadres alleen om uw vraag te beantwoorden.</div>
        </label>
    </form>

Optionele velden

Standaard zijn alle velden in een formulier verplicht, enkel de optionele velden dienen als zodanig gemarkeerd te worden.


    <form action="">
        <label for="test4">
            Uw telefoonnummer <span class="sub-label">(optioneel)</span>
            <input type="text" type="test4">
        </label>
    </form>

Additionele aanwijzingen

Wanneer een invulveld extra aanwijzingen nodig heeft dan dienen deze binnen het label maar na het input element geplaatst te worden.


    <form action="">
        <label for="test5">
            CZ relatienummer <span class="sub-label">(optioneel)</span>
            <input type="text" id="test5">
            <div class="sub-label">U vindt uw relatienummer op uw polis en uw CZ Zorgpas.</div>
        </label>
    </form>

Door de extra aanwijzing op deze manier op te nemen in het label wordt deze automatisch uitgesproken door screenreaders wanneer het invulveld de focus krijgt.
