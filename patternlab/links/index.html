Links in lopende tekst

In deze paragraaf staat een link naar een interne pagina. Maar in deze zin staat een link naar een externe pagina.


    <p>In deze paragraaf staat een link naar een <a href="/consumenten/zorgadvies">interne pagina</a>. Maar in deze zin staat een link naar een <a href="http://www.nu.nl">externe pagina</a>.</p>

Dit zijn de standaard linkjes die in de lopende tekst gebruikt kunnen worden.

Link naar een beveiligde pagina

Er kan worden aangegeven dat aanloggen benodigd is door op het anchor element het attribuut data-type="login" toe te voegen.

Dit is een link naar een pagina binnen de site waarop je moet inloggen

Dit is een link naar een pagine buiten de site waarop je moet inloggen


    <p><a href="/mijncz" data-type="login">Dit is een link naar een pagina binnen de site waarop je moet inloggen</a></p>
    <p><a href="www.digid.nl" data-type="login">Dit is een link naar een pagine buiten de site waarop je moet inloggen</a></p>

Link naar content in andere taal

Het aangeven van de taal waarin de website is geschreven gaat middels het hreflang attribuut van het a-element.

Deze link verwijst naar een Engelstalige website.


    <p><a href="https://www.healthcare.gov/" hreflang="en">Deze link</a> verwijst naar een Engelstalige website.</p>
